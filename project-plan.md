## Goal
Demonstrate how People Operations work can be organized and tracked using GitHub.

## Scope
This project simulates a People Ops initiative focused on improving internal processes and visibility.

## Deliverables
- Repository with structured documentation
- Task tracking system (Issues)
- Workflow board (GitHub Projects)

## Timeline
- Day 1: Repository setup and documentation
- Day 2: Create issues and workflow board
- Day 3: Refine structure and finalize

## Success Criteria
- Clear documentation
- Organized task tracking
- Visible workflow stages
